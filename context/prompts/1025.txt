1.  **Output Format:** Return valid JSON only. No markdown, no explanations.

3.  **`student_answer` (STRING):**

**SORTING RULE:**
- Sort the array numerically by `question_number` (1, 2, 3... 9, 10, 11).
- Do not sort lexicographically (1, 10, 11, 2).

*Input Text:* "Q1. 5 x 5 = 25"
*Output Object:*
```json
{
```
