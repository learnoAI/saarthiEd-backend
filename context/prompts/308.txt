**TASK:**
Extract every question and its corresponding student answer from the image provided. There are approximately 40 questions.

**STRICT JSON SCHEMA:**
You must output a single JSON object with a root key `"questions"` containing an array of objects.
Follow this exact structure:
```json
{
  "questions": [
    {
      "question_number": 1,
      "question": "Q1. 28 , 24",
      "student_answer": "28"
    },
    {
      "question_number": 36,
      "question": "Q36. 25 , 22",

    }

```

    *   Extract the index as a pure **integer**.

2.  **`question` (STRING):**
    *   Format exactly as seen: `Q[Num]. [Val1] , [Val2]`.
    *   **CRITICAL:** Ensure there is a **space before and after the comma** between the two numbers (e.g., `"28 , 24"`).
    *   Flatten vertical text into this single horizontal string.

    *   Extract the handwritten answer found in the blank space/line.
    *   **BLANK HANDLING:** If the student did not write anything, return an empty string `""`.
    *   **WARNING:** Do not infer or calculate the answer. Do not copy numbers from the question into the answer field if the slot is empty. If it looks blank, it is blank.

*   **Sort Order:** The output list **MUST** be sorted numerically by `question_number` from 1 to 40. Do not sort lexicographically (e.g., do not put Q10 after Q1).
*   **Column Layout:** The image may have multiple columns. Read all columns and order them by their Question ID (Q1, Q2, Q3...).

**HANDWRITING TIPS:**
*   Distinguish carefully between handwriting and printed text.
*   Watch for specific digit confusions: **5 vs S**, **0 vs 6**, **1 vs 7**.

Return **ONLY** the JSON object. No markdown formatting, no explanations.
