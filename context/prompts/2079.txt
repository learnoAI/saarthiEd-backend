**TASK:**
Extract every question and its corresponding student answer from the image provided.

**JSON OUTPUT REQUIREMENTS:**
You must output a single JSON object containing a root key `"worksheet_data"`. This key holds an array of question objects.

```json

    }
  ]
}
```

**FIELD FORMATTING RULES (STRICT):**
1.  **`question_number`**:
    - **TYPE:** INTEGER.

    - **FORMAT:** Must start with the label "Q" followed *immediately* by the number and a dot.
    - **CRITICAL:** Remove any space between "Q" and the number.
        - **CORRECT:** "Q1.", "Q12."
        - **INCORRECT:** "Q 1.", "Q 12."

3.  **`student_answer`**:
    - **TYPE:** STRING.
    - Extract exactly what the student wrote. If blank, return `""`.

**HANDWRITING & LAYOUT GUIDELINES:**
- **Digit Recognition:** Distinguish carefully between **9** and **8**, **6** and **0**. Trust the visual stroke evidence.
- **Processing Order:** Sort the array strictly by `question_number` ascending (1 to 40).
- **Whitespace:** Ensure the question string has standard spacing between the label, numbers, and operators (e.g., "Q1. 182 x 100 ="), but **NEVER** inside the "Q[num]" label.
