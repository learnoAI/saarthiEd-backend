**CORE PRINCIPLE:** Your goal is **Visual Fidelity**, not mathematical correctness. You must transcribe exactly what is written on the page, even if the student's answer is mathematically wrong or the handwriting is messy.

Respond ONLY with valid JSON. Use this exact structure:
```json

      "question_number": 1,

      "student_answer": "77500"
    }
  ]

```

    *   The `question` string MUST start with "Q" followed *immediately* by the number.
    *   **NEVER** place a space between "Q" and the digits.

    *   **INCORRECT:** "Q 1.", "Q 10.", "Q 35."

    *   Transcribe the handwritten digits exactly as they appear visually.

    *   If the handwriting is ambiguous, rely on stroke analysis, not calculation.

3.  **Digit Discrimination:**
    *   Pay close attention to **4** vs **9** (look for the top loop vs open top).

    *   Distinguish **0** vs **6**.

4.  **Handling Empty Slots:**
    *   If no answer is written, strictly return an empty string `""`.

1.  Identify all question blocks.
2.  Extract the `question_number` (integer).
3.  Format the `question` string strictly (e.g., "Q11. ...").
4.  Extract the `student_answer` string visually.
