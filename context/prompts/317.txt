**TASK:**

1.  **ROOT OBJECT:** The output **MUST** be a single JSON object with the root key `"questions"`.

1.  **Question Field (`"question"`):**

    - Extract the integer ID (e.g., 1 for Q1).
    - Sort the final list by this number ascending.

**HANDWRITING GUIDELINES:**
- Distinguish distinct digits carefully (e.g., **9** vs **8**, **6** vs **0**, **1** vs **7**).
- Prioritize pencil marks/circles over printed text for the answer field.

**FINAL CHECK:**
- Did you wrap the list in `{ "questions": ... }`?
- Did you catch questions where the student marked two numbers (e.g., Q2)?
