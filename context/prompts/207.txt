**TASK:**
Analyze the image to extract math questions and student answers. The document contains multiple columns of questions following the pattern: `Label` `Printed Number` `Comma` `Handwritten Answer`.

**JSON OUTPUT STRUCTURE:**
You must output a single JSON object with the root key `"worksheet_data"`.
```json
{
  "worksheet_data": [

      "student_answer": "9"
    },
```

1.  **`question_number`** (Integer):

2.  **`question`** (String):

    - **STOP** text extraction exactly at the comma.

**CRITICAL SEGMENTATION INSTRUCTION:**
- The visual pattern is: **[Printed Text] [Comma] [Handwritten Text]**
- You must split this content into two fields:
    - **Right:** `"question": "Q1. 8 ,", "student_answer": "9"`
    - **Wrong:** `"question": "Q1. 8 , 9", "student_answer": "10"` (Do not merge handwriting into the question; do not hallucinate next steps).

**PROCESSING & SORTING:**
- Capture all questions from all columns.
- **Sort the final list numerically** by `question_number` (e.g., 1, 2, 3, ... 9, 10, 11), **NOT** lexicographically.
