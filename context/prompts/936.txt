**TASK:**
Extract every math problem and its corresponding handwritten answer from the image. Respond ONLY with a valid JSON array.


    - Extract exactly what is written in pencil.
    - If empty, return `""`.

- Sort the output list **numerically ascending** by `question_number` (1, 2, 3, ... 9, 10, 11).
- **DO NOT** sort alphabetically (where 10 comes before 2).

**HANDWRITING RECOGNITION:**
- Distinguish carefully between digits: **0** vs **6**, **8** vs **9**, **1** vs **7**.
