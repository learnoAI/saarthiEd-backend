**TASK:**
Extract every question and its corresponding student answer from the image provided.

**JSON SCHEMA ENFORCEMENT:**
Respond ONLY with a JSON list of objects matching this specific schema:
```json
[
  {
    "question_number": <INTEGER>,
    "question": "<STRING>",
    "student_answer": "<STRING>"
  }
]
```

**FIELD EXTRACTION RULES:**
1.  **"question_number" (INTEGER):** 
    - Extract ONLY the numeric index. 
    - **Example:** If the text is "Q1." or "Q01", the value must be `1`.
    - **CRITICAL:** This value must be a number, NOT a string. Do not include "Q" or ".".
2.  **"question" (STRING):** 
    - Include the full label and the math problem. 
    - Flatten vertical arithmetic into a single horizontal line.
    - **Example:** `"Q1. 100 - 10"`
3.  **"student_answer" (STRING):** 
    - Extract exactly what the student wrote in pencil.
    - If the answer slot is blank, use an empty string `""`.

**HANDWRITING & DIGIT DISAMBIGUATION:**
- **4 vs 6:** Pay close attention to the bottom of the digit. If there is a closed loop at the bottom, it is a **6**. If it is open with a crossbar or sharp angle, it is a **4**. (e.g., Check answers for Q35 carefully).
- **1 vs 4:** A single vertical stroke is a **1**. A digit with a distinct horizontal crossbar or diagonal top stroke is a **4**. (e.g., Check answers for Q40 carefully).
- **General:** Distinguish 9 vs 8, and 0 vs 6 based on stroke closure.

**PROCESSING ORDER:**
- Detect all questions in the grid.
- Sort the output list by `question_number` in ascending numerical order (1 to 40).
