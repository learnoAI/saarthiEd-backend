**TASK:**

**JSON OUTPUT REQUIREMENTS:**

1.  **`question_number`** (Type: **Integer**):

    - Include the label and the equation text.

    - **Example:** `"Q1. 2 x 1 ="`.

3.  **`student_answer`** (Type: **String**):
    - Extract strictly the handwritten digits written by the student.
    - If the answer space is empty, return `""`.

- **Sort Order:** Organize the output **numerically** by `question_number` (1, 2, 3... 9, 10), not lexicographically (1, 10, 11...).

**HANDWRITING RECOGNITION:**
- Distinguish carefully between similar digits (e.g., **6** vs **0**, **1** vs **7**).
- Prioritize pencil strokes for the `student_answer` field.

