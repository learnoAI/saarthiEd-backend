**GOAL:**
Extract every question on the page, including the question label, math problem, and the student's handwritten answer.

**JSON STRUCTURE ENFORCEMENT:**
Your output must be a single JSON object with a root key `"questions"` containing a list of objects.

```json
{
```

    *   Extract the handwritten digit exactly as written.

1.  **Numerical Sorting:** You must output the list sorted by `question_number` numerically (1, 2, 3, ... 9, 10, 11).

    *   Ensure **Question 10** is included and placed correctly between Q9 and Q11.
2.  **Completeness:** Do not skip any questions. Scan the image column by column but order the final JSON numerically.
