**TASK:**
Extract every question and its corresponding student answer from the provided image into a structured JSON object.

**STRICT JSON STRUCTURE:**
Your output must be a single JSON object with the root key `"worksheet_data"`, which contains an array of item objects.

**REQUIRED SCHEMA:**
```json
{
  "worksheet_data": [
    {
      "question_number": 1,
      "question": "Q1. 123 + 45",
      "student_answer": "168"
    },
    ...
  ]
}
```

**FIELD RULES:**
1.  **`question_number` (INTEGER):**
    - Extract the numeric ID from the label.
    - **Remove** "Q" and any periods.

2.  **`question` (STRING):**
    - Capture the full text of the question, including the label "Qx.".
    - Flatten vertical math problems into a horizontal string (e.g., "Q1. 747 + 16").
    - Do not allow newline characters within the string.
3.  **`student_answer` (STRING):**
    - Extract the digits written by the student.
    - Return as a string. If the answer area is empty, return `""`.

**HANDWRITING & PROCESSING:**
- Sort the array by `question_number` in ascending order.
- Distinguish ambiguous digits carefully (e.g., 9 vs 8, 6 vs 0, 7 vs 1).
- Ensure the output is valid, parseable JSON.
